What is event delegation in the contex of the DOM, why is it useful?

Instead of attaching a click handler to each item, event delegation allows you to attach a single listener to the parent container. This results in less code and better performance, especially for large sets of elements

How do you manipulate an element's attributes and style using the dom?

add(): Adds a class. remove(): Removes a class. toggle(): Adds or removes a class depending on whether it exists.
contains(): Checks if a class exists.
display: none: Removes the element from the layout flow.
visibility: hidden: Hides the element but keeps its space in the layout.
